# Data Generation

This directory contains public code for VISTA-Track dataset generation.

## Pipelines

- `ue5_mv_generator/`: current preferred UE5-based multi-view rendering pipeline.
- `synthetic_v7_3/`: lightweight Python synthetic pipeline retained for experiments and ablations.

## Public Release Policy

The repository includes source code, configs, and lightweight metadata only.

Not included:

- rendered frames
- generated annotations
- downloaded assets
- UE build artifacts
- caches
- local Python environments
